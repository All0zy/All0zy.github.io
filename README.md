## Project Overview

This repository hosts the source code of a personal academic homepage for Yang Zhang, an undergraduate student in Surveying and Mapping Engineering at Northeastern University (China). The site presents his education, research interests, innovation projects, patents, software copyrights, honors, and leadership experience in a polished and professional layout.

## Technical Stack

- Jekyll
- GitHub Pages
- Markdown-based content management
- SCSS theme customization
- Static assets under the images and files folders

## Repository Structure

- `_pages/` - homepage content written in Markdown
- `_layouts/` - page layout templates
- `_includes/` - reusable components such as sidebar and masthead
- `_sass/` - theme styling and typography settings
- `assets/` - CSS and JavaScript resources
- `files/` - downloadable documents such as CVs and patents
- `images/` - site images and visual materials

## How to Use This Repository

1. Clone or fork this repository.
2. Update the site settings in `_config.yml`.
3. Replace the homepage content in `_pages/about.md` with your own information.
4. Add your own images and PDF files into `images/` and `files/`.
5. Publish the site through GitHub Pages.

## Local Preview

Run the following command locally:

```bash
bash run_server.sh
```

Then open:

```text
http://127.0.0.1:4000
```
